# SKILL.md — 俱乐部老板AI技能包

## 概述

**名称**：俱乐部老板AI助手（Club Owner AI Toolkit）
**版本**：1.0.0
**适用对象**：俱乐部老板、服务型业务创业者、个人IP创业者
**平台**：Claude Code
**语言**：简体中文

## 核心能力

### 1. 内容创作引擎
- 按你的风格自动写短视频逐字稿（五段式结构）
- 图文内容创作（决策框架类）
- 自动应用平台防限流敏感词替换
- 支持抖音/小红书/视频号多平台适配

### 2. 商业决策辅助
- 算账、对比、推荐的决策框架
- 获客渠道选择分析
- 定价策略建议
- 客户分级和转化路径匹配

### 3. 记忆与自动化
- 自动识别和记录重要信息
- 客户跟进状态追踪
- 内容库和选题库管理
- 每日数据统计

## 领域知识覆盖

| 领域 | 内容 |
|------|------|
| 内容创作 | 逐字稿格式、图文格式、选题策略、写作流程 |
| 平台合规 | 抖音敏感词、限流SOP、导流红线、小红书指南 |
| 商业模型 | 俱乐部服务模型、销售分层、客户分型、决策框架 |
| 增长与IP | 账号矩阵、IP打造框架、Vlog方法、爆款本质 |
| 工作流 | 自动记忆、每日自动化 |
