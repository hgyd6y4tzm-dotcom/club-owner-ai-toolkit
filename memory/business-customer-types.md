## 客户类型分类框架

### 一、按动机分类

| 类型 | 特征 | 转化重点 |
|------|------|----------|
| 焦虑驱动型 | 怕落后、怕亏钱 | 降低风险感知，给确定性 |
| 结果驱动型 | 想快速看到回报 | 给路径和案例，展示时间线 |
| 成长驱动型 | 认真想学、愿意投入 | 给体系和方法论 |
| 跟风型 | 看别人做自己也做 | 先劝退不适合的，筛选诚意客户 |

### 二、按状态分类

| 状态 | 特征 | 策略 |
|------|------|------|
| 纯小白 | 没接触过行业 | 先给认知框架，不急着转化 |
| 踩过坑 | 自己试过但亏了 | 共情+方案，转化率最高 |
| 有基础 | 已在做但没放大 | 给增长方案，客单价更高 |
| 老油条 | 什么都懂一点 | 用数据和结果说话 |

### 三、转化路径匹配
- 纯小白 → 免费内容 → 低价入门产品 → 建立信任
- 踩过坑 → 案例展示 → 直接中价产品 → 快速成交
- 有基础 → 深度诊断 → 高价方案 → 长期合作
- 跟风型 → 先筛选 → 不合适的直接劝退

### 四、红牌客户（建议不接）
- 一上来就问"保证能赚多少"
- 要求你先免费服务再付费
- 不尊重服务人员
- 明显超出你能力范围的
