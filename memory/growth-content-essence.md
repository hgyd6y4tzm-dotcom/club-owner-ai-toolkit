## 自媒体爆款四大本质

### 一、是"他媒体"不是"自媒体"
- 内容是给"他"（观众）看的，不是给"自己"看的
- 你关心的 ≠ 观众关心的
- 每发一条之前问：这对观众有什么用？

### 二、内容是产品
- 每条内容都是一次产品交付
- 观众花时间看你的内容 = 付了"注意力货币"
- 你要给等价回报（认知/情绪/实用价值）

### 三、内容的本质是人性
- 好奇、焦虑、贪婪、懒惰、虚荣、从众
- 好内容不是讲道理，是戳人性
- 戳中了，自然传播

### 四、精准匹配供需
- 你的供给（擅长的内容）∩ 观众的需求（关心的话题）= 爆款区
- 只在交集区做内容
- 你擅长但观众不关心 = 自嗨
- 观众关心但你不懂 = 硬蹭

### 五、爆款检查清单
- [ ] 这条内容对观众有什么用？
- [ ] 开头 3 秒能让人停下来吗？
- [ ] 有反直觉/争议点吗？
- [ ] 有人性触发点吗？（好奇/焦虑/共鸣）
- [ ] 结尾让人想做点什么吗？
