## IP 打造框架

### 一、IP 核心公式
- **识别 → 信任**：先让别人知道你、记住你
- **喜欢 → 爱**：再从认可到追随

IP 不是设计出来的，是"真实的你 + 放大的表达"。

### 二、IP 四要素
1. **身份标签**：你是干什么的（一句话说清）
2. **专业背书**：你为什么能干这个（经验/结果/案例）
3. **个性风格**：你说话做事的方式（口语/节奏/态度）
4. **价值观输出**：你相信什么、反对什么

### 三、用户看你内容的 6 个需求
| 需求 | 你的内容怎么做 |
|------|------|
| 好看（审美） | 画面干净、排版舒服 |
| 共鸣（情绪） | 说出他们想说但不敢说的话 |
| 成长（认知） | 给新视角、新框架 |
| 好奇（信息差） | 揭露行业真相、内幕 |
| 情感（陪伴） | 让他们觉得你在跟他们聊天 |
| 解决问题（实用） | 拿来就能用的方法 |

### 四、IP 自查 12 问
1. 你最擅长什么？（能力圈）
2. 你最关心什么？（价值观）
3. 你讨厌什么？（立场）
4. 你的故事是什么？（经历）
5. 别人为什么信你？（背书）
6. 你说话的风格是什么？（表达）
7. 你的目标人群关心什么？（需求）
8. 你和同行的区别在哪？（差异化）
9. 你做过最牛的事是什么？（案例）
10. 你摔过最惨的坑是什么？（共鸣点）
11. 你未来的方向是什么？（愿景）
12. 别人离开你的时候会带走什么？（价值）
