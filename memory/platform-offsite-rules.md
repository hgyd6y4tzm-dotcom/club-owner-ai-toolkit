## 图文违规红线：禁止引导脱离平台

### 一、核心红线
- ❌ 禁止引导用户脱离平台进行交易
- ❌ 禁止引导下载第三方软件
- ❌ 禁止在公开内容中留微信号/手机号/二维码
- ❌ 禁止暗示"去XX平台找我"

### 二、敏感话术替换

| 触发话术 | 替换方案 |
|----------|----------|
| 加我微信 | 后台找我 / 滴我 |
| 扫码加我 | 看主页 |
| 下载XX App | （直接删掉） |
| 去XX平台交易 | 在平台内完成 |
| 私信我报价 | 后台聊 |
| 免费领取 | 后台扣"XX" |
| 点击链接 | （删掉，用主页引导代替） |

### 三、安全表达方式
- "有问题后台滴我"
- "主页有更多内容"
- "评论区扣1，我私你"
- "看主页简介"
- "合集里都写了"

### 四、图文特别提醒
- 图片中不要放二维码/微信号水印
- 聊天截图要打码对方头像和昵称
- 支付截图要打码金额和收款方
