## Vlog 创作方法论

### 一、Vlog 满足的 3 个用户需求
1. **好奇心**：偷窥别人的生活（"他是怎么做的？"）
2. **陪伴感**：一个人在家的背景音（"有个人在说话"）
3. **造梦**：看到自己想要的生活（"我也想这样"）

### 二、平台对 Vlog 的 3 个评价维度
1. **真实性**：有没有真实的生活痕迹，不是剧本感
2. **反差感**：人物前后的变化/冲突/矛盾
3. **互动性**：观众能不能参与进来（评论/跟拍/模仿）

### 三、Vlog 里做 IP 的四步法
1. **展示日常**：让观众看到你的真实状态
2. **嵌入观点**：在日常里自然带出你的判断和看法
3. **制造反差**：从失败到成功、从混乱到有序
4. **给观众角色**：让他们觉得"我在陪你一起做"

### 四、Vlog 标题技巧
- "一个XX人的一天"（身份标签）
- "从XX到XX，我用了X个月"（反差+时间）
- "如果你也在做XX，一定懂我在说什么"（共鸣）
- "做XX的第X天"（连载感）

### 五、Vlog 结尾的金句设计
- 抛一个今天的感受，而不是总结
- 例："走到今天我才明白，XX不是最重要的，YY才是。"
- 不要"谢谢观看""明天见"
