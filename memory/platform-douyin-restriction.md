## 抖音限流处理 SOP

### 触发条件
- 视频审核不通过
- 视频被限制推荐
- 收到违规通知

### 标准流程

**第一步：确认状态**
- 查看系统通知，确认是"拒审"还是"限流"
- 确认具体违规原因

**第二步：分类处理**

| 情况 | 操作 |
|------|------|
| 拒审（不可申诉） | 直接私密该内容，不纠结 |
| 拒审（可申诉） | 修改文案中敏感词 → 私密原内容 → 重发 |
| 限流（可申诉） | 修改后申诉 → 通过则保留，不通过则私密重发 |
| 误判 | 申诉说明内容合规理由，等待复审 |

**第三步：修改策略**
- 检查敏感词（参考 [[platform-douyin-rules]] 替换表）
- 检查是否触碰表达禁区
- 修改后重发，原内容设为私密

**第四步：记录**
- 记录本次违规原因和修改方案
- 更新本地敏感词库

### 核心原则
- **不硬刚平台**：不申诉骂街，不发朋友圈抱怨
- **改了重来**：内容本身没问题，往往是表达触线
- **积累经验**：每次违规都是完善替换表的机会
