## 抖音防限流规则

### 一、敏感词替换表
出稿后自动替换以下词汇：

| 原词 | 替换为 |
|------|--------|
| 赚钱 | 变现 / 有结果 |
| 暴利 | 高利润 |
| 月入X万 | 跑出来的结果 / 做得好的 |
| 保证 | 大概率 / 我们有信心 |
| 100% | 绝大多数 |
| 最好 | 更优的选择 |
| 第一 | 头部 / 前排 |
| 免费 | 零门槛 / 不花钱 |
| 赚钱项目 | 业务方向 |
| 躺赚 | 轻松跑通 |
| 包赚 | 有确定性的 |

### 二、表达禁区
- ❌ 不出现具体收入数字 → ✅ 用模糊描述（"有了结果""跑通了"）
- ❌ 不讨论平台算法/审核机制
- ❌ 不夸大承诺（"保证回本""一定赚钱"）
- ❌ 不引导脱离平台交易
- ❌ 不引导下载第三方软件
- ❌ 评论区不直接留微信号/手机号

### 三、申诉处理 SOP
参考 [[platform-douyin-restriction]]：
- 拒审 → 私密该内容
- 可申诉 → 修改文案后重发，原内容私密
- 不硬刚平台，改了重来

### 四、表情禁区
- "暴富""发财"类表情谨慎使用
- 不用💰🤑等直接暗示金钱的 emoji 在封面
