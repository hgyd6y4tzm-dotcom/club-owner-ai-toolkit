## 每日自动化任务

### 一、每日数据统计
每天自动统计核心业务数据：
- 新增客户数（好友/群）
- 当日收入
- 跟进中客户状态更新

### 二、内容归档
- 当天发布的视频/图文自动记录
- 记录发布平台、发布时间、初始数据
- 3天/7天后回看数据，辅助复盘

### 三、知识提炼
- 从当天的客户对话中提炼新的话术/Q&A
- 从行业信息中提取值得关注的变化
- 存入对应 memory 分类

### 四、推荐工具
- 微信聊天记录导出：可以用 `wechat_history_export` 这个 Claude Code 技能
- 朋友圈内容爬取：定时导出，提炼灵感
- 飞书多维表格：客户跟进和收入追踪
