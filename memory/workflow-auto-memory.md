## 自动记忆系统

### 一、工作原理
AI 助手会在对话中自动识别以下信息并存入记忆：
- 你提出的新规则、新偏好、新工作流
- 你做的决策（选A不选B，附带原因）
- 你纠正的错误（避免再犯）
- 你分享的习惯、工具、账号体系
- 任何"以后都这样"的约定

**不需要你说"记住这个"，AI 自动判断。**

### 二、记忆分类

| 类型 | 内容示例 | 生命周期 |
|------|----------|----------|
| 临时 | 今天的待办、跟进中客户 | 完成后归档 |
| 短期 | 本周选题、本月目标 | 周期结束复盘 |
| 长期 | 内容风格、定价策略、商业模型 | 永久保留，定期检查 |

### 三、文件格式
每条记忆一个 .md 文件，包含：
```yaml
---
name: 短横线命名
description: 一句话概括
metadata:
  type: user | feedback | project | reference
  tags: [标签1, 标签2]
---
正文内容

**Why:** 为什么记录这个
**How to apply:** 怎么用
```

### 四、索引维护
- MEMORY.md 文件维护所有记忆的索引
- 每条记忆一行：`- [标题](文件名.md) — 简述`
- 新记忆自动添加，过期的定期清理
