## 2. 第一步：安装 Node.js

Node.js 是一个基础运行环境。就像你手机要装 iOS 才能装 App 一样，电脑要先装 Node.js 才能装 Claude Code。

### Windows 用户

**2.1** 打开浏览器，访问这个网址：

```
https://nodejs.org/zh-cn
```

**2.2** 你会看到两个绿色大按钮。**点左边那个（LTS）**，不要点右边的。

**2.3** 下载完成后，双击打开安装包。

**2.4** 一路点 **Next（下一步）**，所有选项都用默认的，不要改任何东西。

**2.5** 最后点 **Install（安装）**，等进度条跑完。

**2.6** 验证安装成功：

按键盘上的 **Win 键**（Windows 图标那个键），输入 `cmd`，回车。会弹出一个黑框框。

在黑框框里输入这一行（直接复制粘贴，然后回车）：

```bash
node --version
```

如果看到类似 `v22.12.0` 这样的数字，说明成功了。如果显示"不是内部或外部命令"，重启电脑后再试一次。

### Mac 用户

**2.1** 打开浏览器，访问：

```
https://nodejs.org/zh-cn
```

**2.2** 点左边的绿色 **LTS** 按钮下载。

**2.3** 双击下载的 `.pkg` 文件，一路点继续。

**2.4** 验证安装：按 `Cmd + 空格`，输入 `终端`，回车。在弹出的窗口里输入：

```bash
node --version
```

看到版本号就对了。

## 3. 第二步：安装 Claude Code

Claude Code 是 Anthropic 公司的 AI 助手软件。**你需要先注册一个账号。**

### 3.1 注册 Anthropic 账号

打开浏览器，访问：

```
https://console.anthropic.com
```

用你的邮箱注册。注册后有免费额度，够日常使用。如果用量大，可以绑定信用卡付费（按月扣，不贵）。

### 3.2 安装 Claude Code

打开终端（Windows 是 `cmd` 黑框框，Mac 是「终端」），复制粘贴下面这行，回车：

```bash
npm install -g @anthropic-ai/claude-code
```

等待安装完成（大约 2-3 分钟，取决于网速）。

### 3.3 验证安装成功

在终端里输入：

```bash
claude --version
```

看到版本号就对了。

### 3.4 首次登录

在终端里输入：

```bash
claude
```

第一次运行会提示你登录。会弹出一个浏览器窗口，让你输入 Anthropic 的账号密码。登录成功后，终端里会出现 Claude Code 的界面。

**看到这个界面就说明你成功了：**

```
> 有什么我可以帮你的？
```

输入 `/exit` 回车，先退出来。我们接下来装技能包。

## 4. 第三步：安装本技能包

### 4.1 下载技能包

在终端里，逐行复制粘贴以下命令（每行回车一次）：

```bash
# 先进入你的用户目录
cd %USERPROFILE%
```

```bash
# 下载技能包
git clone https://github.com/hgyd6y4tzm-dotcom/club-owner-ai-toolkit.git
```

> 💡 上面的链接就是真实的下载地址，直接复制粘贴即可。

```bash
# 进入技能包目录
cd club-owner-ai-toolkit
```

### 4.2 安装到 Claude Code

```bash
# 复制主配置文件
copy CLAUDE.md %USERPROFILE%\.claude\CLAUDE.md
```

```bash
# 复制知识库文件
xcopy memory %USERPROFILE%\.claude\projects\club-owner\memory\ /E /I
```

> 💡 **Mac 用户**：把上面命令里的 `%USERPROFILE%` 替换成 `~`，把 `copy` 替换成 `cp`，把 `xcopy ... /E /I` 替换成 `cp -r memory ~/.claude/projects/club-owner/memory/`。

### 4.3 验证安装

在终端里输入：

```bash
claude
```

进入 Claude Code 后，输入：

```
你是谁？
```

如果它回答类似"我是俱乐部老板的AI决策辅助助手"，说明技能包安装成功！

## 5. 第四步：改成你自己的

技能包里的知识和方法论是通用的，但有几个地方需要改成你自己的信息。

### 5.1 打开配置文件

用记事本打开 CLAUDE.md：

**Windows**：打开「文件资源管理器」，在地址栏输入 `%USERPROFILE%\.claude`，回车。找到 `CLAUDE.md`，右键 → 打开方式 → 记事本。

**Mac**：打开访达，按 `Cmd + Shift + G`，输入 `~/.claude`，回车。用文本编辑打开 `CLAUDE.md`。

### 5.2 修改这几行

找到文件末尾的「自定义区」，把下划线替换成你自己的信息：

```
- 你的名字/品牌名：张三（替换为你的名字）
- 你的内容风格关键词：直接、干货、不废话（替换为你的风格）
- 你的核心业务线：XX俱乐部（替换为你的业务）
- 你的主要获客平台：抖音、小红书（替换为你的平台）
- 你的工作文件目录：D:\工作\（替换为你电脑上的实际文件夹）
```

**改完保存，关掉记事本。**

## 6. 第五步：开始使用

### 6.1 日常使用

每次想用 AI 助手的时候：

1. 打开终端（Windows 按 Win 键输入 `cmd`；Mac 打开终端）
2. 输入 `claude` 回车
3. 开始说话

### 6.2 你可以这样说

| 你想做什么 | 你可以这样跟它说 |
|-----------|-----------------|
| 写短视频稿 | "帮我写一条关于XX的逐字稿，3分钟以内" |
| 做决策 | "我现在有两个获客渠道，A和B，帮我算一下选哪个" |
| 记录客户信息 | "今天聊了一个客户，情况是..." （它会自动记） |
| 查之前的记录 | "我之前那个XX客户是什么情况？" |
| 写朋友圈图文 | "帮我把今天的稿子改写成朋友圈版本" |
| 定价建议 | "我这个服务定价多少合适？帮我算算" |

### 6.3 关键快捷键

| 按键 | 作用 |
|------|------|
| `Enter` | 发送消息 |
| 先按 `Esc` 再按 `Enter` | 如果 Enter 不发送（输入法问题），用这个方法 |
| `Ctrl + C` | 中断 AI 输出 |
| `Ctrl + D` | 退出 Claude Code |
| `/exit` | 退出 Claude Code |

> ⚠️ **发送消息不出去的解决方法**：
> 
> 如果你按 Enter 发不出去，99% 是因为中文输入法把 Enter "吃掉了"。
> 
> **解决办法**：打完字 → 切到英文输入法（按一下 `Shift` 键）→ 再按 `Enter` 发送。

## 7. 常见问题

### Q1: 我的数据安全吗？会不会泄露？

**安全。** Claude Code 和本技能包的所有数据都存在你的电脑上。对话内容通过加密传输到 Anthropic 的服务器处理后返回，Anthropic 不会用你的数据训练模型（API 数据保留政策）。

你的客户信息、收入数据、商业策略——全部在你自己的电脑上。

### Q2: 要钱吗？

Claude Code 本身免费。Anthropic 账号有免费额度（每个月几千次对话）。如果用量大，需要付费——大约 $20/月（约 140 元人民币），非常够用了。

本技能包**永久免费、开源**。

### Q3: 我不做俱乐部，做别的生意能用吗？

能。核心框架（内容创作、商业决策、客户管理、平台规则）适用于大多数服务型业务。你可以删除 memory 文件夹里不需要的文件，替换成你自己行业的。

### Q4: 我只做小红书，不用抖音，怎么改？

删除 `memory/platform-douyin-rules.md` 和 `memory/platform-douyin-restriction.md` 即可。AI 助手会自动适应你保留的知识库。

### Q5: 怎么让 AI 更懂我的风格？

多聊。每次你跟它聊的时候，它会自动学习你的偏好并记到 memory 里。**用得越多，它越像你。**

### Q6: 我换了电脑怎么办？

把 `%USERPROFILE%\.claude\` 整个文件夹复制到新电脑的同一位置就行。建议用 U 盘或网盘备份这个文件夹。

### Q7: 出现了我看不懂的英文报错怎么办？

把报错信息复制粘贴给 Claude Code 本身（重新打开一个新的终端窗口），问它"这个报错怎么解决？"。AI 会自己修自己的问题。
